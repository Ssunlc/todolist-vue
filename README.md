# todolist-vue
a todolist by vue   
最近在学习vue.js,这个是前几天练手的demo-一个todolist应用
## 主要功能
回车添加todo，可选中，删除   
下方tab栏筛选active，completed，实时显示未做todo，一键清除已完成
## vue-cli
vue-cli+webpack搭建的项目，header，footer,section(todo)>item+tabs都是单个component
## 主要难点（菜鸟觉得0.0）
- tab栏里的事件，数据传递，以及筛选并反馈到交互
- 子组件向父组件传递数据
